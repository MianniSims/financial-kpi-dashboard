# Financial KPI Dashboard

## Overview
Developed an interactive Tableau dashboard analyzing institutional portfolio KPIs, revenue segmentation, and client risk trends using SQL and Excel-based financial data.

## Tools Used
- Tableau Public
- SQL
- Excel

## Key Metrics
- Total Revenue
- Average Reporting Turnaround
- Portfolio Distribution
- Client Risk Segmentation
